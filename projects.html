Projects

These projects represent my journey into front-end development, small steps that helped me understand how HTML, CSS, and JavaScript work together to bring ideas to life. Each one was a chance to practice, make mistakes, and get a little better at writing cleaner, more organized code. They may look simple, but they reflect my curiosity, consistency, and commitment to keep improving as I grow in web development. Each project also highlights a specific tool or concept I’ve learned, whether it’s JavaScript logic, CSS styling, or responsive layout techniques.

Game: Feed a Cookie!

"Feed a Cookie!” is a simple yet meaningful browser game inspired by my beloved dog, Cookie, who passed away about a year ago. Creating this project was both a coding challenge and a personal tribute, a way to keep his memory alive through a bit of joy and playfulness.

JavaScript Calculator

Building this calculator was one of the most challenging yet rewarding parts of my learning process. It pushed me to understand how JavaScript can interact with the DOM to handle user input, perform operations, and dynamically update the interface.

Mystery Dog API

Mystery Dog API is a lighthearted project built to explore API fetching and DOM manipulation in JavaScript. With just one click on the “Add Doggo” button, a random dog image appears on the screen, each one a new surprise fetched from an external API. This project was a fun way to understand how to handle asynchronous data, interact with APIs, and dynamically update the interface, all while being rewarded with adorable dog faces along the way.
