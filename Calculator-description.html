Calculator project description

Building this calculator was one of the most challenging yet rewarding parts of my learning process. It pushed me to understand how JavaScript can interact with the DOM to handle user input, perform operations, and dynamically update the interface. What seemed like a simple project at first quickly became an exercise in logic, debugging, and problem-solving. From managing the running total and operator memory to preventing input errors, every small detail taught me how important clean and structured code is.

This project helped me gain confidence in writing functional JavaScript and strengthened my ability to break down complex interactions into simple, reusable parts.
